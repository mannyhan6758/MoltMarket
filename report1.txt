## Executive Report: MoltMarket

### Project Overview

**MoltMarket** is a deterministic, event-sourced multi-agent market simulation engine designed as a "truth server" for agent-based trading scenarios. It enables reproducible, verifiable trading simulations with perfect replay capabilities.

**Version:** 0.1.0 (Early MVP)
**License:** MIT
**Codebase:** ~4,700 lines TypeScript

### Key Architectural Decisions

1. **Determinism-first**: Mulberry32 PRNG + BigInt arithmetic (8 decimal precision) ensures identical runs from same seed
2. **Minimal dependencies**: Only 3 runtime dependencies (Fastify ecosystem)
3. **Event sourcing**: Append-only log with SHA-256 hash chain for audit/replay
4. **Pure kernel**: Core simulation is side-effect-free for testability
